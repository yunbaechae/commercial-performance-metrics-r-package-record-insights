# Commercial Performance Metrics Analysis: R Package for Commercial Record Insights

STAT1378 Coding and Communication in Statistics Assignment 2 (2021)

This assignment analyzed Mario Kart 64 world records dataset with a commercial focus on performance metrics, trends, and strategic insights. Developed a custom R package for data processing and reproducible reporting.

## Key Tasks
- Explored records by track, type, shortcut, player, time, and duration  
- Analyzed performance drivers and record trends for benchmarking  
- Built and submitted R package for metrics computation and analysis  

## Skills Demonstrated
- R package development and documentation  
- Data manipulation, trend analysis, and visualization  
- Commercial insights from performance data  
- Reproducible workflows for business applications
